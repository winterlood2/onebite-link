사이드바의 폴더를 클릭했을 때, 그 폴더에 저장된 링크만 보여 주는 페이지를 구현해 줘.

[요구 사항]

- 페이지 주소: /folder/[folderId] 형태의 동적 경로로 만들 것
- 사이드바에서 폴더를 클릭하면 해당 폴더 페이지로 이동하게 할 것
- 메인 영역은 인덱스 페이지처럼 링크 그리드 형태로 표시할 것
- 헤더와 사이드바는 기존 컴포넌트를 그대로 재사용할 것

[주의 사항]

- 화면을 컴포넌트 단위로 적절히 분할해서 구현할 것
- 작업한 결과는 컴포넌트 단위로 보고할 것
- 앞에서 이미 만들어 둔 컴포넌트를 최대한 재사용할 것
